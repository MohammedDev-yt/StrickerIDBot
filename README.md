## рЯФІ Environment Variables (Render / VPS Setup)

You must set these environment variables before running the bot:

- API_ID вЖТ Your Telegram API ID  
- API_HASH вЖТ Your Telegram API Hash  
- BOT_TOKEN вЖТ Bot token from @BotFather  
- OWNER_ID вЖТ Your Telegram numeric ID  
- OWNER_NAME вЖТ Your name (display in bot)  
- START_PIC вЖТ Start image file_id or image URL  
- DB_NAME вЖТ Database file name (example: database.db)  

---
