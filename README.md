## рҹ”§ Environment Variables (Render / VPS Setup)

You must set these environment variables before running the bot:

- API_ID вҶ’ Your Telegram API ID  
- API_HASH вҶ’ Your Telegram API Hash  
- BOT_TOKEN вҶ’ Bot token from @BotFather  
- OWNER_ID вҶ’ Your Telegram numeric ID  
- OWNER_NAME вҶ’ Your name (display in bot)  
- START_PIC вҶ’ Start image file_id or image URL  
- DB_NAME вҶ’ Database file name (example: database.db)  

---

## рҹ“Ң Example (.env format)

API_ID=1234567  
API_HASH=your_api_hash_here  
BOT_TOKEN=123456:ABC-DEF123456  
OWNER_ID=987654321  
OWNER_NAME=YourName  
START_PIC=https://example.com/image.jpg  
DB_NAME=database.db
