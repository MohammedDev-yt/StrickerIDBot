## 🔧 Environment Variables (Render)

You must set these environment variables before running the bot:

- API_ID → Your Telegram API ID  
- API_HASH → Your Telegram API Hash  
- BOT_TOKEN → Bot token from @BotFather  
- OWNER_ID → Your Telegram numeric ID  
- OWNER_NAME → Your name (display in bot)  
- START_PIC → Start image file_id or image URL  
- DB_NAME → Database file name (example: database.db)  

---

## 📌 Example (.env format)

API_ID=1234567  
API_HASH=your_api_hash_here  
BOT_TOKEN=123456:ABC-DEF123456  
OWNER_ID=987654321  
OWNER_NAME=YourName  
START_PIC=https://example.com/image.jpg  
DB_NAME=database.db
